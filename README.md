# Movie Seat Finder

A barebones local web app that searches real Fandango theatres, showtimes, formats, and seat maps. It lets you choose a ZIP/radius, movie, format, date/time range, and a granular seat preference grid, then shows matching results with real Fandango seat maps.

## Run

```bash
python server.py
```

Then open:

```text
http://127.0.0.1:4173/index.html
```
